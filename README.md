# daily-task

一日のタスクを見える可するReactのアプリケーションです。

- ドラッグ＆ドロップでタスクの時間帯を移動することが可能
- text input部分ととタイムライン上のタイルから文字入力可能
- スケジュール、タスク、タイムライン上の情報が同期
- 登録情報はブラウズのローカルストレージに登録（ブラウザのローカルストレージをリセットすると削除される）
